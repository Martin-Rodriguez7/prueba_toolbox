# Frontend Client - CSV Viewer

Aplicación React que consume la API de procesamiento de archivos CSV.

## Requisitos Técnicos

- Node.js 16
- React
- React Bootstrap
- Redux Toolkit

## Instalación y Uso

1. Instalar dependencias:

   ```
   npm install
   ```

2. Iniciar la aplicación (Puerto 3001):

   ```
   npm start
   ```

3. Ejecutar tests:

   ```
   npm test
   ```

## Funcionalidades

- Visualización de datos en tablas ordenadas.
- Filtrado dinámico por nombre de archivo.
- Gestión de estados de carga y errores.
